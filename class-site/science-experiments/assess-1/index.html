← 과학탐구실험 홈

📊 수행평가 1차 - 실험 설계 및 데이터 분석

📅 평가 정보

  • 평가 기간: 2024년 1월 8일 ~ 1월 12일
  • 평가 방식: 개별 실험 및 보고서 제출
  • 배점: 총 100점 (실험 60점, 보고서 40점)
  • 상태: 완료

🎯 평가 목표

  • 과학적 탐구 방법을 이해하고 적용할 수 있다
  • 실험을 설계하고 수행할 수 있다
  • 데이터를 수집하고 분석할 수 있다
  • 결과를 해석하고 결론을 도출할 수 있다

🧪 실험 주제: "물의 온도에 따른 용해도 변화"

실험 재료

  • 소금 (NaCl)
  • 설탕 (C₁₂H₂₂O₁₁)
  • 증류수
  • 비커, 온도계, 저울
  • 가열기구

실험 과정

  1. 물의 온도를 20°C, 40°C, 60°C, 80°C로 조절
  2. 각 온도에서 소금과 설탕의 용해도 측정
  3. 데이터 기록 및 그래프 작성
  4. 결과 분석 및 결론 도출

📈 평가 결과

우수 작품

창의적인 실험 설계와 정확한 데이터 분석을 보여준 학생들의 작품을 소개합니다.

피드백

전체적인 평가 결과와 개선 사항에 대한 피드백을 확인할 수 있습니다.

📚 참고 자료
